Buttons

Buttons need a bit of work if you want them to have a link. Clicking to the right of the button and holding down the mouse button as you move over it until all the text is highlighted will select the entire button for you.

Next you'll see the washed out chain icons at the top of the editor toolbar are darkened as long as the text is highlighted. So leave the text highlighted by only clicking on button with a linked chain. This will open up a menu in which you'll see a section you can paste your URL in. Save that and your button is ready to take people places!

As for the button html code, all you need to change is the class to change the color. The template does have all of the button types ready for you to save some time though.

The Code 

    <button type ="button" class="btn btn-primary"> button text </button>
